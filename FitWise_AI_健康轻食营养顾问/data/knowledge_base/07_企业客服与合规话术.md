# 企业客服与合规话术

## 客服原则

健康食品和健康管理平台客服回答应做到：

1. 基于产品资料和公开健康科普。
2. 不做医疗诊断。
3. 不承诺治疗效果。
4. 不使用绝对化宣传。
5. 对特殊人群提示咨询专业人士。

## 常见问题

### 用户问：吃轻食就一定能瘦吗？

建议回答：

“轻食可以帮助用户更清楚地管理热量和营养结构，但体重变化仍然取决于总热量、运动、睡眠、压力和长期习惯。建议结合个人目标持续记录和调整。”

### 用户问：控糖人群能吃主食吗？

建议回答：

“主食不是绝对不能吃，通常更建议关注份量、种类和搭配方式。可以优先选择糙米、燕麦、红薯、玉米等高纤维主食，并搭配蛋白质和蔬菜。如果已经确诊糖尿病或正在用药，建议咨询医生或注册营养师。”

### 用户问：为什么图片识别的热量可能不准？

建议回答：

“图片识别会受到拍摄角度、遮挡、餐具大小、烹饪用油和实际重量影响，因此结果是估算值。建议用户结合包装营养表、称重或手动修正来提高准确性。”

### 用户问：多久能瘦？

建议回答：

“体重变化与总热量、运动、睡眠、压力和个人基础有关，不能承诺具体速度。更建议关注可持续的饮食结构和生活习惯，多数成年人适合采用温和的体重管理节奏。”
